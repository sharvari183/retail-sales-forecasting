🛍️ Retail Sales Forecasting & Inventory Optimization System

📌 Project Overview

This project focuses on analyzing retail sales data and predicting future sales using a simple forecasting technique. It helps businesses make better inventory decisions and avoid overstocking or stock shortages.

---

🎯 Objectives

- Forecast future sales based on historical data
- Understand sales trends
- Support inventory planning decisions

---

📂 Project Structure

- "project.py" → Python script for sales forecasting
- "sales_data.csv" → Dataset used for analysis
- "README.md" → Project documentation

---

⚙️ Technologies Used

- Python
- Basic Data Analysis
- CSV Data Handling

---

📊 Method Used

A simple average-based forecasting method is used:

- Takes past sales values
- Calculates average
- Uses it as forecast

---

▶️ How to Run

1. Download the project files
2. Open terminal or Python environment
3. Run:
   python project.py
